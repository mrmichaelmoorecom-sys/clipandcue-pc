# clipandcue for PC

Clipboard history for Windows (Linux later) that pastes back **editable**.
clipandcue snapshots every format on the clipboard, so an Office shape comes
back a shape and an Illustrator path comes back vector art, not a flattened
screenshot of one.

Part of the **clip…** family with [clipandcue for Mac](https://github.com/mrmichaelmoorecom-sys/clipandcue)
and [clipandnote](https://github.com/mrmichaelmoorecom-sys/clipandnote).
This is a ground-up native rewrite (Tauri + Rust), not a port.

## Status

🚧 Pre-development. The project brief lives in [CLAUDE.md](CLAUDE.md).

## Principles

- Tiny and native-feeling. No Electron.
- Local only. No accounts, no telemetry, no server.
- Free for personal use (CC BY-NC 4.0).
